### 生成步骤

``` bash
# 安装依赖
yarn

# 以开发模式运行在 2018 端口
yarn dev

# 生成应用程序
yarn build


# 单元测试和端到端测试
yarn test

# 对 src 文件夹下的 Vue/Js 文件进行语法检查
yarn lint

```

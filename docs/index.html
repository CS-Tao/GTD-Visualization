Electron

Electron 是一个使用 JavaScript, HTML 和 CSS 等 Web 技术创建原生程序的框架，它负责比较难搞的部分，你只需把精力放在你的应用的核心上即可。

Vue

Vue是一套用于构建用户界面的渐进式框架。与其它大型框架不同的是，Vue 被设计为可以自底向上逐层应用。Vue 的核心库只关注视图层，不仅易于上手，还便于与第三方库或既有项目整合。

Element-UI

Element-ui 是由饿了么前端出品的一套 Vue.js 后台组件库，组件清爽美观，与 Vue 2.0 紧密集成。

# 项目生成步骤

# 安装依赖
yarn

# 以开发模式运行在 2018 端口
yarn dev

# 生成应用程序
yarn build


# 单元测试和端到端测试
yarn test

# 对 src 文件夹下的 Vue/Js 文件进行语法检查
yarn lint
